If the PuTTY client does not automatically take you to the application directory path, please follow these steps to configure it.

Once configured, you will be redirected to the correct path upon clicking 'Connect SSH'.

This configuration only needs to be set up once and will apply to all future environment connections made from MXevolve Studio.

Open WinSCP > go to Options > Select Preferences > Integration > Applications > Under PuTTY/Terminal client path add the following command:

{{ puttyClientConfiguration }}
